## URL schemes

## application  - Heavily coupled with specific application program
## browser      - Used by Web browsers for internal use or extensions
## conflict     - There are multiple independent definitions or usages
## contact      - Contact address such as email address or tel number
## ill-formed   - Syntactically broken as URL scheme name
## local        - Resource in user's local environment
## registerable - whitelisted schemes <http://c.whatwg.org/#whitelisted-scheme>
## scriptable   - May contain insecure data within the URL
## web-core     - Important URL schemes of the Web platform
## wildcard     - Wildcard "*" is used for URL scheme name
